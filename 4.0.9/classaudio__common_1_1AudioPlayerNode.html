Detailed Description

ROS 2 node that subscribes to stamped audio messages and plays them through a PortAudio output device.

One PortAudio stream is opened per unique combination of sample format, sample rate, and channel count. Mono/stereo channel conversion is performed automatically when the incoming channel count differs from the configured output channel count.

ROS 2 Parameters
  • channels (int, default 2): Number of output audio channels.
  • device (int, default -1): PortAudio output device index. -1 selects the system default output device.
Subscriptions
  • audio (audio_common_msgs/msg/AudioStamped, SensorDataQoS)
