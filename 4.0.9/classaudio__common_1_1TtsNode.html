Detailed Description

ROS 2 node that converts text to speech using espeak and streams the resulting audio via a ROS 2 action server.

When a TTS action goal is received the node invokes espeak to produce a temporary WAV file, then reads and publishes the file as AudioStamped messages at the file's natural playback rate. Per-chunk feedback is published so clients can track progress. Any active goal is aborted when a new goal is accepted.

ROS 2 Parameters
  • chunk (int, default 4096): Number of float samples per published message.
  • frame_id (string, default ""): TF frame ID attached to published message headers.

void TtsNode::execute_callback ( const std::shared_ptr< GoalHandleTTS > goal_handle)
private

Executes a TTS goal in a background thread.

Synthesises speech with espeak, reads the resulting WAV file in chunks, publishes each chunk as an AudioStamped message, and sends per-chunk feedback. Sets the goal result to the synthesised text on success.

Parameters
goal_handleHandle to the goal being executed.
