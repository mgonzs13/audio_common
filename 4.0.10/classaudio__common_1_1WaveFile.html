Detailed Description

Lightweight reader for 16-bit PCM WAV files.

Opens a WAV file, parses its header to extract audio metadata, and provides sequential reading of sample data converted to normalised float values in the range [-1.0, 1.0]. Only 16-bit PCM encoding is supported.

Typical usage
WaveFile wf("/path/to/file.wav");
if (!wf.open()) { /* handle error *\/ }
* std::vector<float> buf;
* while (wf.read(buf, 512)) {
* // process buf ...
* }

bool WaveFile::open ( )

Open the WAV file and parse its header.

Reads the RIFF/WAV header to populate sample_rate_, channels_, and bits_per_sample_, then seeks to the start of the PCM data section.

Returns
true on success, false if the file could not be opened or the header is not a valid RIFF/WAV header.

◆ read()

bool WaveFile::read ( std::vector< float > & buffer,
size_t size )

Read the next size frames from the file into buffer.

Samples are converted from int16_t to float in the range [-1.0, 1.0] using int16ToFloat(). buffer is resized to size × channels_ elements.

Only 16-bit PCM WAV files are supported; the call returns false immediately for any other bit depth.

Parameters
[out]bufferDestination vector that receives the converted samples.
[in]sizeNumber of frames (not bytes) to read.
Returns
true if exactly size frames were read, false on end-of-file, read error, or unsupported bit depth.
